# Aura v2.2.1 更新说明

## 2026-06-12 项目记忆体验优化

- 项目记忆现在会在你一段时间没有继续提问后自动整理，不再出现长时间等待但没有生成记忆的情况。
- 顶部状态栏的“项目记忆”弹窗更好用了：点击记忆路径可以直接打开记忆目录。
- 弹窗里的按钮改为“立即整理”，你可以手动触发当前项目的记忆整理。
- 立即整理完成后，弹窗会刷新最新整理时间、整理状态和待整理数据数量。
- 记忆整理会尽量按一次完整任务来处理，减少把同一个任务拆成前后几段导致总结不连贯的问题。
- 待整理内容很多时，Aura 会分批整理并持续处理到完成，不会因为等待时间较长就提前提示超时。
- 修复了部分记忆文件生成后内容为空的问题，项目经验、排查记录和用户偏好会更稳定地写入对应文件。
